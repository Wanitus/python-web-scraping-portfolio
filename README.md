# Python Web Scraping Portfolio

This repository contains several Python web scraping projects built using BeautifulSoup and Requests.

Projects

1. Quotes Scraper
Scrapes quotes and authors from quotes.toscrape.com

2. Books Scraper
Extracts book titles and prices from books.toscrape.com

3. Ecommerce Scraper
Scrapes laptop product data from webscraper.io demo site

Tech Stack

- Python
- Requests
- BeautifulSoup
- Pandas

Run

pip install -r requirements.txt
python scraper.py
